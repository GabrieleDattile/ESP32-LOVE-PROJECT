# ESP32 Love Project ❤️

> *An interactive, romantic, and customizable digital gift powered by an ESP32 and a TFT display.*

This project uses an ESP32 to render beautiful, smooth animations including a beating parametric heart, cute chibi bears holding roses, fluttering butterflies, and a Pac-Man chasing love pills! With the press of a button, it cycles through these animations and displays randomized, customizable romantic messages. 

*Perfect for Valentine's Day, anniversaries, or just to surprise someone special.*

## ✨ Features

* **Custom Initials & Beating Heart:** A stunning, mathematically generated parametric heart that pulses, featuring the initials of you and your loved one.
* **Smooth Animations:** * 🐻 Cute bears hopping with roses.
  * 🦋 Butterflies with realistic flapping mechanics.  
  * 👾 Pac-Man and Ms. Pac-Man animations.
* **Interactive Messages:** Press a button to reveal a random romantic message enclosed in an elegant UI.
* **Ambient Particles:** A continuous background effect of falling hearts and colorful particles.
* **Hardware Interactivity:** Uses two push buttons to easily cycle through states and messages.

---

## 🛠️ Hardware Requirements

* **ESP32 Development Board** (This code is optimized for boards with built-in displays like the *LilyGO T-Display* or *T-Display S3*, but works with standalone ESP32s).
* **TFT Display** (ST7789 controller recommended, the code is set up for a custom init sequence and uses a 280x160 resolution).
* **2x Push Buttons** (If your board doesn't have them built-in).
